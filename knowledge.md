# Project Knowledge - Unity Asset Toolkit 
 
Unity Asset Store project for AI/pathfinding tools. C#, Unity 7 LTS, Windows dev environment. 
 
## Quickstart 
```bash 
# Open Unity Hub, add project from assets/ folder 
# Open scripts in Visual Studio 2022 
``` 
 
## Architecture 
- docs/ - Documentation and guides 
- notes/ - Planning and research 
- prototypes/ - Quick C# tests 
- assets/ - Unity projects 
- scripts/ - Build tools 
 
## Conventions 
- C# naming: PascalCase classes, camelCase methods 
- Unity 7 LTS target 
- Visual Studio 2022 as IDE 
- Asset Store submission guidelines 
 
## Development Environment 
- Unity 7 LTS - installed via Unity Hub 
- Visual Studio 2022 - primary IDE 
- Git 2.45.1 - version control 
- Windows 11
