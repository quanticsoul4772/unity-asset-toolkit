# SwarmAI - Multi-Agent Coordination System

SwarmAI is a Unity asset for creating intelligent swarm behaviors and multi-agent coordination.

## Features

- Multi-agent state machines
- Coordinated movement and formations
- Resource gathering behaviors
- Threat detection and avoidance
- Flocking and swarming algorithms

## Quick Start

1. Add a `SwarmManager` component to an empty GameObject
2. Add `SwarmAgent` components to your agent prefabs
3. Configure behaviors in the Inspector
4. Press Play!
