# Development Environment Setup  
  
## Required Software  
- Unity 6 LTS (2025.x or later)  
- JetBrains Rider OR Visual Studio 2022  
- Git + Git LFS  
- Windows 10/11  
  
## Unity Installation  
1. Download Unity Hub from unity.com  
2. Install Unity 6 LTS with Windows Build Support  
3. Add Android/iOS modules if targeting mobile
